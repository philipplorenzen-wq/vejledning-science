Check the course description to see how you pass re-exams in courses with ongoing tests or exam prerequisites.  


Ways of taking re-exams in courses with ongoing tests

If you need to retake an exam in a course with ongoing tests, you have 3 options:

  1. Check the course description to see if re-exams have a different exam form and register for the re-exam if that is the case. 
  2. Ask the course coordinator whether it's possible for you to take an ordinary exam in a course with ongoing tests without re-registering for the actual course.
  3. Register for the course again if there are no other options.

Find information about the re-exam and how to contact your course coordinator in your course description


Ways of taking re-exams in courses with examination prerequisites

If you meet the examination prerequisites, you can register for the re-exam in either the re-exam period or an ordinary exam period.


If you don't meet the examination prerequisites, check the course description to see whether it's possible to meet the examination prerequisites without taking the course again. If it's possible, you'll need to agree with the course coordinator how you can meet the prerequisites before the re-exam.

Read your course description


If it isn't possible for you to meet the prerequisites before the re-exam, you have to re-register for the course in order to pass it.
