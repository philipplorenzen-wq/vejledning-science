You have used an exam attempt and can then decide whether you want to: 

  • retake the exam in the same project
  • start over with a new project. 

Retake the exam in the same project 

If you want to retake the exam in the same project, you must:

  • agree with your supervisor on when you will restart the project
  • register for the same project
  • submit your report again – possibly in a revised version that remains within your problem formulation
  • take the oral exam. 

Start over with a new project 

If you choose to do a new project, you must follow the same procedure as last time:

  • Find a new topic and possibly a company.
  • Find a new supervisor, if necessary.
  • Complete and submit a new project contract.
