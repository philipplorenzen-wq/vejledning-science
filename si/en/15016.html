You have to do three things:

  1. Contact the University of Copenhagen's information security unit by sending an email to securitybreach@adm.ku.dk so that you can get guidance on reporting the event to the Data Protection Agency and guidance on informing the affected persons.

  2. Report the security incident to the Data Protection Agency within 72 hours after you have discovered it.

  3. Inform the persons whose data accidentally have been disclosed.

What is a security incident?

If you lose personal data, eg because you collect them on questionnaires that are lost, it is a security incident.

It is also a security incident if you accidentally send personal data to others who do not have a right to receive them or you unintended publish personal data on the internet.
