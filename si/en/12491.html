At UCPH, you have access to the Sensus Access program if you have a reading or visual impairment. Here, you can make your study materials available in different formats. For example, a word document can be converted to an audio file that you will receive by email.

How to use the tool

  1. Go to the tool 
  2. Upload your file 
  3. Choose a format 
  4. Receive a file in your email
