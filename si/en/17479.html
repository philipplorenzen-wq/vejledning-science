Can I register for a course I’ve previously been registered for?

Original side på KUnet: https://kunet.ku.dk/study/study_content/Pages/Can-I-register-for-a-course-I%E2%80%99ve-previously-been-registered-for-SCI.aspx

You can re-register for a course if

  1. you did not pass the exam
  2. the course is not fully subscribed.


If the course has a limited number of places, you will be bottom of the list when the places are allocated – unless you need to attend classes to take the exam (e.g. some courses with examination prerequisites or ongoing tests).


If you were ill for the exam last time

You are entitled to a place on the course if you were ill for the exam last time, and class attendance is a requirement for taking the exam.

You must register during the ordinary registration period to get your guaranteed place.

If you are not entitled to a place on the course, because class attendance is not a requirement for taking the exam, you are only entitled to a new place if you have withdrawn from the course due to illness.
