What about my exam attempts when my course is replaced by another one?

Placering: Courses and teaching › Course changes › Discontinued courses

Sprog: English · Elementtype: Accordion (narrow format)

Gælder for målgrupperne: Det Natur- og Biovidenskabelige Fakultet - Kandidat

Original side på KUnet: https://kunet.ku.dk/study/study_content/Pages/What-about-my-exam-attempts-when-my-course-is-replaced-by-another-one-SCI.aspx

Whether or not exam attempts which you have used to try to pass your discontinued course are transferred to the new course depends on what you are to replace the course with.


Identical courses – exam attempts count

Exam attempts already used to try to pass the discontinued course count in the identical course.

For instance, if you have used 2 exam attempts, but not passed the discontinued course, the exam in the identical course will count as your 3rd exam attempt.


Replacement courses (equivalent courses) – 3 new exam attempts

You have 3 new exam attempts to complete a replacement course. Any exam attempts that you have used to try to pass the discontinued course do not count when you have to pass a replacement course.


New self-selected course – three new exam attempts

When self-selecting a new elective or restricted elective course, you have 3 new exam attempts to complete the new course.
Any exam attempts that you have used to try to pass the discontinued course do not count towards passing the new course.
