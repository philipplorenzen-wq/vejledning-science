You get a free UCPH-license for Microsoft 365. With the license you can use OneDrive, Word, Excel, PowerPoint, Outlook, Teams and Copilot Chat etc. while you are an active student. 

You can use the programs online at m365.cloud.microsoft/apps or install them on your computer or tablet.

See guidance on how to install the programs on your computer.

Your data will not be shared
When you use the Microsoft 365 applications through UCPH's license, your data is protected from being shared with third parties or used to train Microsoft's AI models.

Important points:

  • You may install Microsoft 365 on up to five devices.
  • If you are both a student and an employee at UCPH, other rules apply - Read more in the Employee Guide.
  • You need to use multi-factor authentication to access Microsoft 365 applications. The multi-factor method you have set up to access your KUmail is the same one you will use when prompted to authenticate in Word, Excel, Outlook, etc.
On the Help with login page, you can read about all the ways you can use multi-factor authentication, if you haven’t already set it up.

When you leave UCPH 

Your license – including access to OneDrive – closes when you finish your degree programme. You'll need to take a back-up of any documents and data you want to save before then. Read more about the agreement in the Software Library.
