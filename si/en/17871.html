If you fall ill just before or in the run-up to your oral exam, you must 

  1. give notice of absence from the exam due to illness 
  2. retake the exam in the same project or start from scratch. 

You will not be using an exam attempt if you give notice of absence from the exam due to illness. 

Take the exam in the same project 

If you want to take the exam in the same project, you must

  • agree with your supervisor on when you will restart the project
  • register for the same project
  • submit your report (again) – possibly in a revised version that stays within your problem formulation
  • if you have chosen an oral exam: attend it as the final step.

Start over with a new project 

If you choose to start a new project, you must follow the same procedure as last time:

  • Find a new topic and possibly a company. 
  • Find a new supervisor, if necessary.
  • Complete and submit a new project contract.
