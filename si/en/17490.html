I’ve failed an exam with multiple examinations. Do I need to retake them all?

Check your course description to see if you need to retake all examinations again or only the examination(s) you are missing:

  • Unless stated otherwise, you must retake all examinations – and you must retake them in the same exam period.
  • If stated in your course description that you can pass the different examinations in different exam periods, you should not retake the examinations you have already passed.


If an examination includes a written paper

If one of your examinations includes a written paper, which is not assessed individually, you can – unless otherwise stated in the course description – choose to either resubmit a revised paper or submit a new paper for the re-exam.
