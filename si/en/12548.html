How do I access my KUmail?

Placering: IT and facilities on campus › IT and support › KUmail

Sprog: English · Elementtype: Accordion (narrow format)

Gælder for målgrupperne: Københavns Universitet - Alle uddannelsesniveauer

Original side på KUnet: https://kunet.ku.dk/study/study_content/Pages/Webmail-login.aspx

  1. Go to webmail.ku.dk or click on "KUmail" on KUnet´s frontpage.
  2. Log in using your UCPH username followed by @alumni.ku.dk and password. 
    Note: If you are both a student and an employee at KU, you must use @ku.dk.
  3. Multi-factor authenticate yourself with one of the MFA methods below. To do this, you must first register your selected method.
    Please note: Even if you are already using one of these methods - for example for your private email - you still have to register your KU account to be able to use the method for webmail.ku.dk. See how in these guides:
  • Multi-factor Authentication via NetIQ app
  • Multi-factor Authentication via Microsoft Authenticator (if you have to reset your UCPH login in the app and have MitID: use this guide.)
If you have not got a smartphone:
  • Multi-factor-authentification via SMS code (NOTE! If you are employed at the University - e.g., as a student assistant - you cannot use this method)
  • Multi-factor Authentication via Yubikey app

    • Multi-factor Authentication via Yubikey emergency code
